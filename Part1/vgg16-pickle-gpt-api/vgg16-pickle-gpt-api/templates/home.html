1. Présentation du Projet

Cette application web est développée avec Flask et déploie plusieurs modèles de Machine Learning dans une seule interface. Les modèles sont chargés une seule fois au démarrage pour améliorer les performances.

Modèles Déployés

  • Classification d’image (VGG16) – Route /predict
  • Modèle de régression (Pickle) – Route /regpredict
  • Génération de texte (GPT-2) – Route /textgen

Sécurité et Stockage

  • Utilisation de secure_filename pour sécuriser les uploads.
  • Vérification des extensions des fichiers images.
  • Volume Docker pour la gestion des fichiers temporaires.

2. Déploiement avec Docker

Structure du Projet

Le projet contient : app.py, Dockerfile, docker-compose.yml, requirements.txt, model.pkl et le dossier templates/.

Commande de Déploiement

docker-compose up --build -d

Accès à l’Application
